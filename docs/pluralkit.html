what's a pluralkit?

Pluralkit is a Discord bot we use for system management. Thanks to its (very well documented) API, we can display all sorts of handy, rapidly changing information! How neat is that? This page is dedicated to all of the neat Things™ we've made using it.

what is this meant to... do, exactly?

Basically, we log information about who is actively existing at any given moment (because DID is a fickle thing, and many idiots inhabit this meat-suit), and also information about each of said idiots. As a result, you can browse that information in an accessible* format.

*it's not really accessible and the javascript is really broken but at least you don't need to use a discord bot
